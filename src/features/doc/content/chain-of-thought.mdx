## Features

- Collapsible reasoning panel built on shadcn/ui Collapsible.
- Step statuses for complete, active, and pending reasoning work.
- Search result cards with source badges and external links.
- Image attachments with captions for visual evidence.
- Step-type icons with support for custom icon components.
- Context-aware compound components powered by React Context.
- Fade and slide content transitions with reduced-motion support.
- Responsive layout for narrow and wide surfaces.

## Usage

```tsx
import {
  ChainOfThought,
  ChainOfThoughtContent,
  ChainOfThoughtSearchResults,
  ChainOfThoughtStep,
  ChainOfThoughtSteps,
  ChainOfThoughtTitle,
  ChainOfThoughtTrigger,
} from "@/components/chain-of-thought"
```

```tsx
<ChainOfThought defaultOpen>
  <ChainOfThoughtTrigger>
    <ChainOfThoughtTitle>Reasoning</ChainOfThoughtTitle>
  </ChainOfThoughtTrigger>
  <ChainOfThoughtContent>
    <ChainOfThoughtSteps>
      <ChainOfThoughtStep status="complete" title="Search sources">
        <ChainOfThoughtSearchResults results={results} />
      </ChainOfThoughtStep>
    </ChainOfThoughtSteps>
  </ChainOfThoughtContent>
</ChainOfThought>
```
