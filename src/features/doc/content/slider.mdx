## Features

- Supports controlled and uncontrolled value management.
- Supports left-to-right and right-to-left horizontal layouts.
- Provides compound parts for tracks, ranges, thumbs, marks, and labels.
- Keeps form integration available with a hidden input.

## Usage

```tsx
import { Slider } from "@/components/slider"
```

```tsx
<Slider defaultValue={[48]} max={100} min={0}>
  <Slider.Track>
    <Slider.Range />
    <Slider.Mark value={0} />
    <Slider.Mark value={50} />
    <Slider.Mark value={100} />
  </Slider.Track>
  <Slider.Thumb aria-label="Intensity" />
  <Slider.HiddenInput name="intensity" />
</Slider>
```

### RTL

```tsx
<Slider defaultValue={[48]} dir="rtl" max={100} min={0}>
  <Slider.Track>
    <Slider.Range />
    <Slider.Mark value={0} />
    <Slider.Mark value={50} />
    <Slider.Mark value={100} />
  </Slider.Track>
  <Slider.Thumb aria-label="Intensity" />
</Slider>
```

## Composition

Use the following composition to build a `Slider`.

```text
Slider
├── Slider.Track
│   ├── Slider.Range
│   └── Slider.Mark
│       ├── Slider.Tick
│       └── Slider.Label
├── Slider.Thumb
└── Slider.HiddenInput
```
