## Features

- Wraps Liveline with a compound layout for headers, metrics, chart body, and footer content.
- Shares chart data through context so values, deltas, and legends stay in sync.
- Supports Liveline options including windows, reference lines, momentum, and value overlays.
- Follows light and dark mode by default while allowing explicit theme overrides.

## Usage

```tsx
import { LiveChart } from "@/components/live-chart"
```

```tsx
<LiveChart data={data} value={value} color="#22c55e" showValue>
  <LiveChart.Header>
    <div>
      <LiveChart.Title>Conversion pulse</LiveChart.Title>
      <LiveChart.Description>
        Rolling checkout activity from the last few minutes.
      </LiveChart.Description>
    </div>
    <LiveChart.Metric>
      <LiveChart.Value formatter={(currentValue) => `${currentValue}%`} />
      <LiveChart.Delta />
    </LiveChart.Metric>
  </LiveChart.Header>
  <LiveChart.Body>
    <LiveChart.Chart />
  </LiveChart.Body>
  <LiveChart.Footer>
    <span>Live feed</span>
    <span>Target 62%</span>
  </LiveChart.Footer>
</LiveChart>
```
