## Features

- Wraps Liveline with a compound layout for headers, metrics, chart body, and footer content.
- Shares chart data through context so values, deltas, and legends stay in sync.
- Supports Liveline options including windows, reference lines, momentum, and value overlays.
- Follows light and dark mode by default while allowing explicit theme overrides.

## Usage

```tsx
import { LiveChart } from "@/components/live-chart"
```

```tsx
<LiveChart data={data} value={value} color="#22c55e" showValue>
  <LiveChart.Header>
    <div>
      <LiveChart.Title>Conversion pulse</LiveChart.Title>
      <LiveChart.Description>
        Rolling checkout activity from the last few minutes.
      </LiveChart.Description>
    </div>
    <LiveChart.Metric>
      <LiveChart.Value formatter={(currentValue) => `${currentValue}%`} />
      <LiveChart.Delta />
    </LiveChart.Metric>
  </LiveChart.Header>
  <LiveChart.Body>
    <LiveChart.Chart />
  </LiveChart.Body>
  <LiveChart.Footer>
    <span>Live feed</span>
    <span>Target 62%</span>
  </LiveChart.Footer>
</LiveChart>
```

## API Reference

### LiveChart

Wraps the chart and provides Liveline data to the compound components.

| Prop       | Type                            | Description                                  |
| ---------- | ------------------------------- | -------------------------------------------- |
| `data`     | `LivelinePoint[]`               | Array of time/value points.                  |
| `value`    | `number`                        | Latest value for the animated line.          |
| `theme`    | `"system" \| "light" \| "dark"` | Chart theme. Defaults to `"system"`.         |
| `children` | `ReactNode`                     | Compound chart sections.                     |
| `...props` | `Liveline props`                | Props passed to the underlying Liveline API. |

### LiveChart.Chart

Renders the Liveline canvas using the nearest `LiveChart` data and options.

| Prop        | Type              | Description                                      |
| ----------- | ----------------- | ------------------------------------------------ |
| `data`      | `LivelinePoint[]` | Optional chart data override.                    |
| `value`     | `number`          | Optional current value override.                 |
| `className` | `string`          | Additional classes for the Liveline container.   |
| `...props`  | `Liveline props`  | Optional Liveline overrides for this chart slot. |

### Layout Components

`LiveChart.Header`, `LiveChart.Metric`, `LiveChart.Body`, `LiveChart.Footer`, and `LiveChart.Legend` extend the native `div` element.

| Prop       | Type        | Description                                   |
| ---------- | ----------- | --------------------------------------------- |
| `children` | `ReactNode` | Optional custom content for the section.      |
| `...props` | `div props` | Props passed to the rendered section element. |

### Text Components

`LiveChart.Title` extends `h3`, `LiveChart.Description` extends `p`, and `LiveChart.Value` / `LiveChart.Delta` extend `span`.

| Prop        | Type                               | Description                                |
| ----------- | ---------------------------------- | ------------------------------------------ |
| `formatter` | `(value: number) => ReactNode`     | Optional value formatter.                  |
| `value`     | `number`                           | Optional value override.                   |
| `...props`  | `h3`, `p`, or `span` element props | Props passed to the rendered text element. |
