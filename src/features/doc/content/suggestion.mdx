## Features

- Renders suggestions as shadcn/ui buttons with variant and size props.
- Emits the selected suggestion string through each button's click callback.
- Wraps on narrow viewports while preserving horizontal scrolling when needed.
- Accepts any `ScrollArea` props on the suggestions container.

## Usage

```tsx
import { Suggestion, Suggestions } from "@/components/suggestion"
```

```tsx
const suggestions = [
  "Draft a concise project update",
  "Compare server actions and API routes",
  "List edge cases for this flow",
]
```

```tsx
<Suggestions>
  {suggestions.map((suggestion) => (
    <Suggestion
      key={suggestion}
      suggestion={suggestion}
      onClick={(value) => console.log(value)}
    />
  ))}
</Suggestions>
```
