## Features

- Supports controlled and uncontrolled value management.
- Clamps changes with optional minimum and maximum values.
- Handles decimal increments with step-aware rounding.
- Keeps a compact icon-only surface with accessible labels.

## API Reference

### Stepper

Extends the native `div` element and accepts these additional props.

| Prop             | Type                      | Description                                |
| ---------------- | ------------------------- | ------------------------------------------ |
| `value`          | `number`                  | Controlled value.                          |
| `defaultValue`   | `number`                  | Initial value for uncontrolled usage.      |
| `min`            | `number`                  | Minimum allowed value.                     |
| `max`            | `number`                  | Maximum allowed value.                     |
| `step`           | `number`                  | Amount changed by each button press.       |
| `disabled`       | `boolean`                 | Disables both controls.                    |
| `onValueChange`  | `(value: number) => void` | Called when either control changes value.  |
| `decrementLabel` | `string`                  | Accessible label for the decrement button. |
| `incrementLabel` | `string`                  | Accessible label for the increment button. |
