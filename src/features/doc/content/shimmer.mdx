## Features

- Animates a shimmer sweep across text with a looping linear transition.
- Supports custom animation duration and gradient spread.
- Renders as any element through the `as` prop.
- Adapts to light and dark themes with local CSS variables.

## API Reference

### Shimmer

Extends the selected element through the `as` prop and accepts these additional props.

| Prop        | Type          | Description                                   |
| ----------- | ------------- | --------------------------------------------- |
| `children`  | `string`      | Text content to apply the shimmer effect to.  |
| `as`        | `ElementType` | Element or component to render.               |
| `className` | `string`      | Additional classes for the rendered element.  |
| `duration`  | `number`      | Duration of the shimmer animation in seconds. |
| `spread`    | `number`      | Spread multiplier based on the text length.   |
