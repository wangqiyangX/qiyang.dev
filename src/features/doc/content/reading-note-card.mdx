- Compact layout for reading notes and article highlights.
- Accepts title, source, excerpt, metadata, and tags.
- Uses local design tokens for light and dark mode.
- Keeps styling self-contained and easy to adapt.

## Usage

```tsx
import { ReadingNoteCard } from "@/components/reading-note-card"
```

```tsx
<ReadingNoteCard
  title="Notes should be small enough to keep moving"
  source="Component note"
  excerpt="A compact card for showing a reading note with context, summary, and lightweight tags."
  meta="May 2026"
  tags={["component", "notes", "ui"]}
/>
```

## API Reference

### ReadingNoteCard

Extends the native `article` element and accepts these additional props.

| Prop      | Type       | Description                       |
| --------- | ---------- | --------------------------------- |
| `title`   | `string`   | Main note title.                  |
| `source`  | `string`   | Small label for the note source.  |
| `excerpt` | `string`   | Short summary shown in the card.  |
| `meta`    | `string`   | Compact metadata, such as a date. |
| `tags`    | `string[]` | Optional tags rendered as badges. |
