## Features

- Wraps Liveline with a compound layout for headers, metrics, chart body, and footer content.
- Shares chart data through context so values, deltas, and legends stay in sync.
- Supports Liveline options including windows, reference lines, momentum, and value overlays.
- Follows light and dark mode by default while allowing explicit theme overrides.

## Usage

```tsx
import { LiveLineChart } from "@/components/live-line-chart"
```

```tsx
<LiveLineChart data={data} value={value}>
  <LiveLineChart.Header>
    <div>
      <LiveLineChart.Title>Conversion pulse</LiveLineChart.Title>
      <LiveLineChart.Description>
        Rolling checkout activity from the last few minutes.
      </LiveLineChart.Description>
    </div>
    <LiveLineChart.Metric>
      <LiveLineChart.Value formatter={(currentValue) => `${currentValue}%`} />
      <LiveLineChart.Delta />
    </LiveLineChart.Metric>
  </LiveLineChart.Header>
  <LiveLineChart.Body>
    <LiveLineChart.Chart color="#22c55e" showValue />
  </LiveLineChart.Body>
  <LiveLineChart.Footer>
    <span>Live feed</span>
    <span>Target 62%</span>
  </LiveLineChart.Footer>
</LiveLineChart>
```

## Composition

Use the following composition to build a `LiveLineChart`.

```text
LiveLineChart
├── LiveLineChart.Header
│   ├── LiveLineChart.Title
│   ├── LiveLineChart.Description
│   └── LiveLineChart.Metric
│       ├── LiveLineChart.Value
│       └── LiveLineChart.Delta
├── LiveLineChart.Body
│   └── LiveLineChart.Chart
└── LiveLineChart.Footer
    └── LiveLineChart.Legend
```
