## Features

- Renders Mermaid chart source as SVG on the client.
- Matches light and dark shadcn themes with Mermaid theme variables.
- Shows loading and parse-error states with shadcn/ui primitives.
- Accepts Mermaid config overrides for diagram-specific needs.
