## Features

- Tracks pointer coordinates relative to the component root.
- Replaces the native pointer with a rectangular cursor.
- Draws horizontal and vertical guide lines through the pointer origin.
- Uses a compound component API so the content, guides, and cursor can be composed independently.

## Usage

```tsx
import { CrosshairCursor } from "@/components/crosshair-cursor"
```

```tsx
<CrosshairCursor>
  <CrosshairCursor.Content>
    <div>Your surface content</div>
  </CrosshairCursor.Content>
  <CrosshairCursor.Guides />
  <CrosshairCursor.Cursor />
</CrosshairCursor>
```

## Examples

### Position Change

```tsx
const [position, setPosition] = React.useState({ x: 0, y: 0 })

<CrosshairCursor onPositionChange={setPosition}>
  <CrosshairCursor.Guides />
  <CrosshairCursor.Cursor />
</CrosshairCursor>
```

### Custom Rectangle

```tsx
<CrosshairCursor>
  <CrosshairCursor.Guides />
  <CrosshairCursor.Cursor className="border-primary" width={32} height={20} />
</CrosshairCursor>
```
